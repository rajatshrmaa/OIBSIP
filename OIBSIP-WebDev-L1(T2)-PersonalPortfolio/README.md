## 📖 Description

This is a responsive Personal Portfolio Website developed using HTML, CSS, and JavaScript. It showcases my introduction, skills, education, certificates, featured projects, and contact information in a modern and professional user interface.

The portfolio is designed with a clean dark theme, smooth layouts, responsive design, and interactive sections to provide an engaging user experience across different devices.

- Responsive Navigation Bar
- Hero Section
- About Me Section
- Skills Section
- Education Timeline
- Certificates Section
- Featured Projects
- Contact Section
- Social Media Links
- Responsive Design
- Modern Dark Theme UI
